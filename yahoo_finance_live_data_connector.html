The One-Line Fix — copy into your dashboard's fetchYahoo()
// BEFORE — fails due to CORS: const url = `https://query1.finance.yahoo.com/v8/finance/chart/${ticker}?interval=1d&range=${days}d`; const r = await fetch(url, {headers:{'Accept':'application/json'}}); // AFTER — works in any browser via free proxy (no key needed): const yahooUrl = `https://query1.finance.yahoo.com/v8/finance/chart/${ticker}?interval=1d&range=${days}d`; const proxy = `https://api.allorigins.win/raw?url=${encodeURIComponent(yahooUrl)}`; const r = await fetch(proxy);
Also replace the hardcoded CHART array with the real prices returned by fetchYahoo('SPY', 252) — that gives you a full year of real closing prices to drive the history chart at actual dollar values (e.g. $480–$570), not a synthetic index.
