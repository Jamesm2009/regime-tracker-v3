How to Read This Chart
Green line — Growth
SPY 20-day return. Above 0% = growth rising → favour equities. Below 0% = growth falling → risk-off. The further from zero, the stronger the signal.
Amber line — Inflation
DBC+GLD average 20-day return. Above 0% = inflation rising → favour commodities & energy. Below 0% = inflation falling → favour bonds & tech.
Blue dashed — Gate (63d)
DBC+GLD 63-day return. Above 0% = inflation still elevated (≈CPI>3%) → Q4 deflation signals are blocked. Below 0% = gate clear, Q4 can activate.
Regime quadrant = where the two solid lines sit relative to zero.  Both above → Q2 Overheating.  Green above, amber below → Q1 Goldilocks.  Both below → Q4 Deflation (if gate also below).  Green below, amber above → Q3 Stagflation.

① Signal Calculation (daily)
1
Growth proxy — SPY rolling 20-day return. Positive = growth rising. Coincident with what ETFs respond to.

Inflation proxy — Average of DBC + GLD rolling 20-day return. Positive = inflation expectations rising.
3
Inflation gate — DBC + GLD rolling 63-day return. Positive ≈ sustained CPI above ~3%. Used only to filter false Q4 signals.

③ 3-Day Confirmation Filter
A
A regime change is flagged when today's quadrant differs from yesterday's. No action yet.
B
The flag is confirmed only if the same quadrant holds on day+1 and day+2. Eliminates 42% of false signals.
C
Unconfirmed flags are ignored — the previous regime remains active. Alerts only fire on confirmed changes (~40/year).
④ Q4 Inflation Gate
A
If a Q4 signal fires but the 63-day DBC+GLD momentum is still positive, inflation hasn't truly rolled over — it's a false deflation read.
B
Action: skip — hold the previous regime. 63% of historical Q4 signals were filtered this way.
C
When gate clears (63d momentum negative): act with XLU · XLP · XLV · IEF · IEI · SHY. Hit rate rises from 32% to 53%.
