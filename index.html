① Signal Calculation (daily)
1
Growth proxy — SPY rolling 20-day return. Positive = growth rising.
2
Inflation proxy — Average of DBC + GLD rolling 20-day return. Positive = inflation rising.
3
Inflation gate — DBC + GLD rolling 63-day return. Positive = sustained elevated CPI (≈ >3%). Used only to filter Q4.

③ Confirmation Filter (3 days)
A
A regime change is flagged when today's quadrant ≠ yesterday's.
B
It is confirmed only if the same quadrant holds on day+1 and day+2. Cuts false signals by 42%.
C
Unconfirmed flags are ignored — the previous regime remains active. No action taken.
④ Q4 Inflation Gate
A
If a Q4 signal fires but the 63-day DBC+GLD momentum is still positive, the signal is a false deflation read.
B
Action: skip — hold the previous regime. 63% of historical Q4 signals were filtered this way.
C
When the gate is clear (63d momentum negative): act with XLU · XLP · XLV · IEF · IEI · SHY. Hit rate rises to 53%.
