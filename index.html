Methodology

68 ETFs  ·  Apr 2021–Apr 2026  ·  1,259 trading days  ·  215 raw regime change events  ·  120 confirmed events after 3-day filter  ·  Forward 20-day return measured from signal date  ·  Q4 events during elevated 63-day commodity momentum skipped (26 of 41 events filtered)  ·  Q4 basket: XLU · XLP · XLV · IEF · IEI · SHY

① Signal Calculation (daily)
1
Growth proxy — SPY rolling 20-day return. Positive = growth rising. Simple, coincident, and directly tied to what ETFs respond to.
2
Inflation proxy — Average of DBC + GLD rolling 20-day return. Positive = inflation expectations rising. Commodity composite beats single-asset signals in backtest.
3
Inflation gate — DBC + GLD rolling 63-day return. Positive ≈ sustained CPI above 3%. Used exclusively to filter false Q4 (Deflation) signals.
② Quadrant Assignment
Q1
Goldilocks — Growth ↑  Inflation ↓  →  Tech, Growth equities. QQQ · IWF · MGK · SPY · XLK · SMH · XLC · IWM
Q2
Overheating — Growth ↑  Inflation ↑  →  Commodities, Cyclicals. XLE · XLB · DBC · GLD · SLV · USO · XLI · PAVE
Q3
Stagflation — Growth ↓  Inflation ↑  →  Gold, Energy, Short bonds. GLD · SLV · USO · DBC · XLE · SHY
Q4
Deflation — Growth ↓  Inflation ↓  →  Defensives, Mid bonds (gate must be clear). XLU · XLP · XLV · IEF · IEI · SHY
③ 3-Day Confirmation Filter
A
A regime change is flagged on day D when today's quadrant differs from yesterday's quadrant. No action yet.
B
The flag is confirmed only if the same quadrant holds on day D+1 and D+2 as well. This eliminates 42% of false signals with minimal return cost.
C
Unconfirmed flags are ignored — the previous regime remains active. No rebalancing. Alerts only fire on confirmed changes (~40 per year historically).
④ Q4 Inflation Gate
A
If a Q4 signal is confirmed but the 63-day DBC+GLD momentum is still positive, it is a false deflation read — inflation hasn't truly rolled over.
B
Action: skip — hold the previous regime position instead of rotating into defensive bonds. 63% of historical Q4 signals were filtered this way.
C
When the gate clears (63d momentum negative): act with XLU · XLP · XLV · IEF · IEI · SHY. Hit rate rises from 32% to 53%.
⑤ Live Daily Updates — How to Connect Yahoo Finance
→
Yahoo Finance endpoint — Five free tickers needed: SPY, DBC, GLD, IEF, SHY. URL format: https://query1.finance.yahoo.com/v8/finance/chart/SPY?interval=1d&range=100d
Returns close prices for last 100 days. No API key required.
→
Calculation — From close prices, compute: SPY 20d return (growth), avg DBC+GLD 20d return (inflation), avg DBC+GLD 63d return (gate). Compare to zero. 5 lines of logic. Runs in under 1 second.
→
Automation at 5pm CT — A scheduled script (GitHub Actions free tier, AWS Lambda, or a home server cron job at 0 23 * * 1-5 UTC) fetches data, recalculates, and writes a JSON file. This dashboard reads from that file on load.
→
Alert layer — When a new confirmed regime is detected (day 3 of a new quadrant), send an email or Slack webhook. ~40 alerts per year. No action needed until an alert fires.
Note on this demo: Yahoo Finance blocks direct browser requests from sandboxed iframes (CORS policy). In production, route the fetch through a lightweight backend (Node.js, Python Flask, or even a Cloudflare Worker) that adds the appropriate headers. The button above will attempt a live fetch and fall back to embedded data automatically.
